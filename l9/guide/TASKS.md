1. implementar el sistema (backend), debe tener 5 microservicios:
• Servicio de Pedidos.
• Servicio de Inventario.
• Servicio de Facturación.
• Servicio de Transporte.
• Servicio de Notificaciones.
los microservicios pueden ser aplicaciones de flask/fastapi, deben estar todos dockerizados en un compose

2. implementar el sistema (frontend), esperar a que esté el backend y diseñar las suficientes pantallas para hacer uso de todo el codigo del backend

3. diseñar y ejecutar los casos de prueba funcionales, las ejecuciones son con captura cada una y basandose en lo que pide la guia

4. configurar y ejecutar jmeter o k6, basandose en lo que pide la guia

5. el informe y todo lo que es texto plano xd
